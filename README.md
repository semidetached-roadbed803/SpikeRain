## 📋 What is SpikeRain?

SpikeRain is a tool that cleans rain from single photos. It uses a new method based on spiking neural networks. These networks work like simple models of the brain to improve images. SpikeRain was introduced at WACV 2026, a computer vision conference.

This program helps you get clearer photos even if they were taken in the rain. It works on pictures you have saved on your computer.

## 🧩 How SpikeRain Works

SpikeRain uses spiking neural networks. These networks mimic how brain neurons send signals. This method is efficient and helps remove rain drops without losing image details.

It handles different types of rain patterns, like streaks or drops, and cleans photos faster than older methods.
